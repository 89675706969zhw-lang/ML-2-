Постановка задачи
Тема: Прогнозирование вероятности выпадения осадков на следующий день (RainTomorrow) в Австралии.

Цель исследования:
Разработать и оптимизировать модель машинного обучения для бинарной классификации, способную с высокой точностью прогнозировать вероятность дождя на завтрашний день (RainTomorrow = Yes/No) на основе метеорологических наблюдений.

Данные:
Используется набор данных (Dataset), содержащий историю ежедневных погодных наблюдений с различных метеостанций Австралии. Целевая переменная — RainTomorrow. Датасет соответствует требованиям индивидуального задания (смешанные типы признаков, >20 атрибутов, >20 тыс. записей).

План работы
План составлен в строгом соответствии с реализованным программным кодом (pipeline, optuna, bootstrap):

1. Разведочный анализ данных (EDA) и оценка качества данных:

Загрузка данных и первичный осмотр структуры.
Генерация автоматического отчета с помощью библиотеки ydata-profiling для выявления пропусков, экстремальных значений (выбросов), корреляций и типов распределения признаков.
2. Предобработка данных (Data Preprocessing):

Разделение данных на обучающую и тестовую выборки.
Построение автоматизированного конвейера (Pipeline):
Обработка пропусков: Заполнение медианой для числовых признаков и модой для категориальных.
Кодирование: Применение метода WOEEncoder (Weight of Evidence) из библиотеки category_encoders для преобразования категориальных признаков.
Масштабирование: Стандартизация числовых признаков с помощью StandardScaler.
3. Моделирование и оптимизация (Modeling & Optimization):

Выбор трех алгоритмов классификации:
Логистическая регрессия (Logistic Regression) — как базовый линейный алгоритм.
Случайный лес (Random Forest) — ансамблевый метод бэггинга.
LightGBM (LGBMClassifier) — эффективный метод градиентного бустинга.
Настройка гиперпараметров с использованием фреймворка Optuna для максимизации метрики ROC-AUC.
Использование стратифицированной кросс-валидации (StratifiedKFold) для устойчивой оценки моделей.
4. Оценка качества и интерпретация (Evaluation & Interpretation):

Анализ важности признаков с помощью метода переcтановок (Permutation Importance).
Сравнение моделей по метрике ROC-AUC.
Построение ROC-кривых с 95% доверительными интервалами, рассчитанными методом Bootstrap (бутстрэп), для статистической оценки надежности прогнозов.
