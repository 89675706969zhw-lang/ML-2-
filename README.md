про индивидуальную задачу

заключается в решении похожей задачи бинарного моделирования предрасположенности, то есть прогнозируем вероятность некоторого события
у каждого свой датасет, до 15 января подобрать себе датасет и согласовать его с пуклиным
датасеты искать в: 
1. попросить perplexity
2. ods.ai
3. kaggle
4. google datasets
5. олимпиада дано (библиотека материалов - задания прошлых лет - заключительный тур - базы данных

параметры индивидуального датасета
1. не менее 20 атрибутов (признаков)
2. смешанный тип (категориальные признаки, так и непрерывные признаки)
3. число записей от 20 тыс.
4. могут  быть метки обучающей выборки, а меток тестовой выборки нет - в этом случае обучающий датасет сами делим на обучающую и тестовую
5. датасеты не должны повторяться

1. решать задачу можно в питоне или в логином+питоне 
2. провести разведочный анализ данных и оценка качества данных с помощью библиотеки ydata-profiling (можно и другие, но надо чтобы библиотека развивалась)
нужно смотреть пропуски, экстримальные значения 
3. строим 3 многофакторных модели (логрегрессия + 2 на наш выбор), применяя кроме WoE-кодирования ещё какую-то схему из category_encoders 
4. метрики: roc-auc, доверительны интервалы для кривой и индекса
